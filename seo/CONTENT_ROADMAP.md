# Relocation Manager SEO System

## Site taxonomy

- **Home:** brand + conversion + trust
- **Resources hub:** editorial SEO entry point
- **Pillar pages:** one deep page per major intent
- **How-to pages:** task-based search intent
- **State pages:** trucking laws / parking / permits by state
- **Equipment pages:** box truck, hotshot, flatbed, lowboy, conestoga, power-only
- **Audience pages:** owner-operators, dispatchers, brokers, carriers, moving companies
- **Trust pages:** fraud prevention, cargo insurance, broker setup, FMCSA/DOT basics

## Page templates

1. **Pillar page**
   - 1 H1, 4–8 H2s, FAQ section, internal links, schema, conversion CTA

2. **How-to guide**
   - problem → steps → checklist → mistakes → tools → CTA

3. **State page**
   - rules, permits, scale houses, parking, routing cautions, local links

4. **Comparison page**
   - use cases, pros/cons, pricing notes, ideal user, FAQs

5. **Resource hub page**
   - links to the best content by audience and intent

## Publishing rules

- Every page gets one primary keyword theme.
- Every page links upward to a hub and sideways to 3–6 related pages.
- Every page uses a unique title, description, H1, FAQ, and CTA.
- Every page should answer a real search intent, not just repeat phrases.
