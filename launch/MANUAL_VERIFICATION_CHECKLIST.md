# Manual Verification Checklist — Stage 1

## Driver / Carrier

- [ ] Name
- [ ] Phone/email
- [ ] Home city / pickup radius
- [ ] Equipment type: 15, 16, 20, 24, 26 ft box truck, liftgate, ramp, cargo van, truck+trailer, hotshot
- [ ] Photos of vehicle/equipment if available
- [ ] DOT/MC if applicable
- [ ] Insurance status if applicable
- [ ] Preferred lanes
- [ ] Minimum rate per mile
- [ ] No-touch / driver-assist preferences

## Customer / Broker / Shipper

- [ ] Name/company
- [ ] Phone/email
- [ ] Website or business proof if available
- [ ] Payment method readiness
- [ ] Planned load volume
- [ ] Load types
- [ ] Pickup/delivery regions
- [ ] Willing to accept customer ratings
- [ ] Starter or Pro plan interest

## Authentication / Access

- [ ] Valid login
- [ ] Invalid login
- [ ] Forgot password
- [ ] Create account
- [ ] Remember me
- [ ] Session expiration
- [ ] Logout
- [ ] Mobile responsiveness
- [ ] Role redirects
- [ ] Subscription validation
- [ ] No extra charge after logout/sign-in
- [ ] Screen reader labels and error announcements
- [ ] Keyboard navigation and visible focus states
- [ ] High-contrast readability on mobile and desktop

## Red flags

- mismatched contact info
- asks to move off-platform immediately
- vague load details after follow-up
- payment terms unclear
- refuses identity/business verification
- repeated complaints
