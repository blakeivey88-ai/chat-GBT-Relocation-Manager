# Pricing — Relocation Manager

## Tier 1 — Customer Pickup Request

**$9.99/month**

For customers that need loads picked up from their office, warehouse, shop, or place of work.

Includes:

- direct load request intake
- Ralph helps clean up pickup/load details
- verified brokers/carriers can see the request
- customer rating profile
- post loads and pickup requests, but no claiming other people's loads

## Tier 2 — Independent Driver

**$29.99/month**

For individual self-insured drivers / owner-operators who work for themselves.

Includes:

- equipment profile
- saved pickup lanes
- Ralph load-fit help
- matching load alerts
- verified driver profile
- can browse and claim loads

## Tier 3 — Broker / Fleet Access

For brokers and fleets by truck count:

- **1–3 trucks:** $59.99/month
- **4–7 trucks:** $79.99/month
- **7–12 trucks:** $149.99/month

Includes:

- ability to view verified customer direct load requests
- respond to demand when they have a truck or can make it worth covering
- truck capacity profile
- saved customer lanes / demand visibility
- future broker dashboard access
- can claim loads and post covered freight
